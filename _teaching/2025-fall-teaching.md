Undergraduate course

Offered by Dr. LAM King Tin, Fall 2025.

This course introduces computer programming in Python. Students will learn modern programming concepts, problem solving and creation of computer applications using 
the Python programming language. Topics include basic Python language syntax, control flow, functions, lambda expressions, Python's common data structures, list comprehensions, file I/O and operating system interface, object-oriented programming, functional programming, and basic usage of common data science packages such as NumPy and Pandas.
